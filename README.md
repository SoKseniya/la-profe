# La profe — методист по самостоятельному изучению языков (скилл для Claude Code)

**La profe** — это скилл для [Claude Code](https://claude.com/claude-code): спокойный, дотошный методист-наставник по иностранным языкам. Он не ведёт уроки — он **строит и пересматривает план самостоятельного обучения**: от живого онбординга до детального расписания по дням с конкретными, проверенными ресурсами.

Под капотом — доказательная методика овладения языком (SLA): понятный вход чуть выше уровня, обязательный речевой выход, словарь из собственного чтения, распределённые повторения, зона посильной трудности. La profe не отливает план по шаблону — он каждый раз выводит его из вашей цели, уровня и интересов.

## Что он делает

- **`setup`** — новый план: онбординг (уровень по навыкам, цель, время, интересы, учебники) → маршрут к цели по месяцам в часах → детальный план текущего месяца по дням.
- **`review`** — пересмотр после блока: сверяет прогресс, пересчитывает срок до цели, переписывает следующий блок на новых ресурсах.
- **`adjust`** — лёгкая подкрутка текущего блока, если он не зашёл (слишком трудно/скучно/времени меньше).

План сохраняется в обычные markdown-файлы в папке, которую вы выберете.

## Чего он НЕ делает

- Не ведёт уроки и не объясняет грамматику в чате (это работа ресурсов плана или живого преподавателя).
- Не делает карточки (это работа Anki или другого SRS — La profe только говорит, что туда занести).
- Не следит за регулярностью занятий (это трекер привычек).

## Установка

Скилл = папка `la-profe/` (файл `SKILL.md` + `references/`), которая кладётся в каталог скиллов Claude Code `~/.claude/skills/`.

**Самый простой способ — дайте Claude Code ссылку на этот репозиторий и попросите установить скилл.** Например, скажите ему:

> Установи мне скилл по этой ссылке: https://github.com/SoKseniya/la-profe — склонируй репозиторий и скопируй папку `la-profe` в `~/.claude/skills/`.

**Вручную** (то же самое, копипастой в терминал):

```bash
git clone https://github.com/SoKseniya/la-profe.git /tmp/la-profe-public
mkdir -p ~/.claude/skills
cp -r /tmp/la-profe-public/la-profe ~/.claude/skills/la-profe
```

После этого перезапустите Claude Code (или начните новую сессию) — скилл подхватится. Проверить: спросите Claude Code «есть ли у меня скилл la-profe?».

## Использование

В Claude Code:

```
la-profe setup испанский      # построить новый план
la-profe review испанский     # пересмотреть после блока
la-profe adjust испанский     # подкрутить текущий блок
```

Можно и просто описать словами: «хочу составить план изучения испанского» — Claude подхватит скилл сам.
